二、核心设计理念
2.1 设计一句话
"吞噬万物，自由进化，拥抱或反抗你的蟹化宿命。"
2.2 核心设计支柱
支柱
说明
吞噬即成长
吃是一切行为的起点，所见皆可食
形态即玩法
每一次进化选择都实时改变外观与机制
生态即对手
AI驱动的动态生态系统，不是割草是生存
蟹化即选择
用真实的生物学梗"趋同进化"构建核心策略分支
2.3 情绪曲线设计
开局（弱小迷茫）→ 第一次进化（惊喜发现）→ 中期成型（策略成型）→ 遭遇天敌（紧张逃亡）→ 蟹化抉择（关键分支）→ 终极形态（视觉震撼）→ 死亡/通关（进化回放，分享欲望）

3.2.4 蟹化机制（核心特色）
蟹化值（Carcinization Score）：
- 每选择一个甲壳类进化，蟹化值 +1~3（依节点而定）
- 蟹化值达到 5 时，触发 "初阶蟹化"：DEF+5 HP+20 SPD+3，侧向移动加速，水中呼吸
- 蟹化值达到 10 时，触发 "完全蟹化"：DEF+8 ATK+10 HP+30，帝王之钳，范围伤害+抓取
- 蟹化值达到 15 时，触发 "超蟹化"：DEF+12 ATK+12 HP+50 SPD-5，完全蟹化形态，解锁隐藏结局线索
反蟹化路线：
- 选择非甲壳类进化，蟹化值不会增加
- 蟹化值 = 0 且已解锁 ≥ 6 个进化时，触发"原始纯净"：SPD+15 ATK+8
- 混合路线（如"章鱼+翅膀+真菌"）可以解锁独特的"混沌形态"(s_chaos)
- 隐藏进化"灯塔水母"需要零蟹化值 + 在深海生存5分钟才能解锁

3.3 生态系统 AI
3.3.1 生物行为状态机
每个生物拥有以下状态：
┌─────────┐    发现猎物    ┌─────────┐    进入范围    ┌─────────┐
│  游荡   │ ────────────→ │  狩猎   │ ────────────→ │  攻击   │
│(Wander) │                │ (Hunt)  │                │(Attack) │
└────┬────┘                └────┬────┘                └────┬────┘
     │ 发现更强天敌              │ 猎物逃脱                  │ 击杀
     └────────────────────────→ │ ←─────────────────────────┘
                                 ▼
                          ┌─────────┐
                          │  逃跑   │
                          │ (Flee)  │
                          └────┬────┘
                               │ 脱离危险
                               ▼
                          ┌─────────┐
                          │  恢复   │
                          │(Recover)│
                          └─────────┘
说明：
- WANDER：随机方向移动，每1.5~3.5秒切换方向；肉食/杂食生物在仇恨范围内发现玩家时切换到HUNT
- HUNT：朝目标移动，进入攻击范围（atkRange + 双方半径）时切换到ATTACK；超出栓绳距离时放弃返回WANDER
- ATTACK：冷却结束时执行performAttack()造成伤害；HP低于20%时切换到FLEE
- FLEE：远离玩家，距离足够远时进入RECOVER
- RECOVER：短暂停留后恢复WANDER
- 夜间仇恨范围 ×0.7 倍系数，生物刷新量 ×1.5 倍

四、游戏模式
4.1 主线模式：进化之旅（story）
目标：在30分钟内击败4个区域Boss。
流程：
1. 选择初始基因组合（通过基因库局外解锁）
2. 出生在地图中心安全区域（7×7格草地）
3. 吞噬→进化→变强
4. 击杀生物推进进度条，在20%/40%/60%/80%进度节点触发Boss战
5. 击败全部4个Boss即为通关
6. 通关后解锁无尽模式

进度系统（ProgressTimeline）：
- 击杀目标总数：80只
- 4个Boss节点分别在 20%（16杀）、40%（32杀）、60%（48杀）、80%（64杀）触发
- Boss按群系分配：雨林→森蚺女王、沙漠→沙海蠕虫、苔原→霜巨人蟹、深海→克拉肯

4.2 无尽模式（endless）
移除30分钟时间限制
持续刷新生物（最多同屏45只）
成就挑战：存活60分钟解锁"永恒猎手"

- 无胜负条件
